# Health Insurance Fraud Detection

## Dataset
Kaggle link: (https://www.kaggle.com/datasets/mahfuz95/synthetic-health-insurance-claims-fraud-detection)

## Objective
Predict fraudulent claims using supervised learning.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- SVM

## How to Run
1. Open notebooks in Kaggle or Jupyter
2. Run each notebook sequentially (RQ1 → RQ7)

## Outputs
- Tables saved as CSV
- Figures saved as PDF
